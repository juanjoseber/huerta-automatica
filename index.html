Justificación del Proyecto

La automatización de huertas permite un uso más eficiente del agua, mejora la productividad y facilita el monitoreo del estado de las plantas. Este proyecto busca ofrecer una solución accesible y tecnológica para personas que deseen cultivar sus propios alimentos de forma sostenible.

Posibles Sensores a Emplear

  • Sensor de humedad del suelo (ej. YL-69)
  • Sensor de temperatura y humedad DHT11 o DHT22
  • Sensor de lluvia
  • LDR (sensor de luz) para control de iluminación
  • Sensor ultrasónico para control de nivel de agua
